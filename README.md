## キャリアプロフィール

私はSEO、リスティング広告などを手掛けるインターネット広告コンサルタント会社にて、PHPで作られた社内インフラツールの保守/運用を行い、新規事業である勤怠交通費清算アプリの開発や、営業支援ツールの開発を行っておりました。  
PHPの処理の書き方を調べたりするとき、Rubyの記事も多くヒットすることからRubyにも興味を持ち、独学でRuby, Ruby on Railsを学びました。

その後Rubyを使って開発をしている会社にて、Ruby on Railsを使った単品リピート通販のシステム開発を行い、そこではリーダ長としてDev/Opsチームの立ち上げも行い、エンジニアのマネジメントも行いました。  
その他、Amazon Connectを使った自動架電システムの作成や、Amazon ConnectについてAmazon Web Service様の開催するイベントにスピーカーとして登壇。  
機能開発などだけでなく、リリースフローをGitHub Flowに変更し、GitHub Actionsを用いてリリース頻度が週に1〜2回だったものを週5〜7に引き上げするなど、改善施策などの開発も行っていました。

バックエンドエンジニアとしてRuby on Railsでの開発がメインではありますが、Reactを用いたフロントエンドの開発、GitHub Actionsでのリリース、レビューアサインの自動化など、幅広い開発経験があります。

現在では、オンライン診察のシステム開発を行っています。

コミュニケーション能力の高いエンジニアとして、チームビルディングや採用など幅広い範囲の業務を行うことができ、開発においてはビジネス職である営業やCSチームと直接コミュニケーションを取ることで本当にユーザが必要としているシステムを作れることを強みにしております。
